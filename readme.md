* 使用方式
```bash
curl -fsSL https://raw.githubusercontent.com/co2f2e/onekey-ops/master/server_init.py -o server_init.py
sudo python3 server_init.py
```
* 功能

| 序号 | 功能             |
|------|----------------|
| 1    | 修改 SSH 端口      |
| 2    | 系统更新和清理        |
| 3    | 安装并启用 UFW      |
| 4    | 防火墙端口管理        |
| 5    | 清除系统日志和缓存      |
| 6    | 查看或修改当前时区      |
| 7    | 安装 Nginx       |
| 8    | 卸载 Nginx       |
| 9    | 安装 Nftables    |
| 10   | 卸载 Nftables    |
| 11   | 自动优先选择 IPv6 连接 |
| 12   | 修改当前用户登录密码     |
| 0    | 退出             |

* 适用环境
Ubuntu / Debian 系列
